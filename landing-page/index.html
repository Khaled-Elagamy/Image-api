Landing Page

    Welcome to the image resizer program

    Here how to use the program to resize your image

    Put this in the url (Follow the syntax)

    "/api/image/resize?name={image name}&width={image width}&height={image height}"

    Available Images

    Here the name of the images available
    choose one from the avaliable images

    API documentation

    Endpoint:/resize
    Resize the image you want

    Reset:/reset
    Delete the thumbnail folder after you finish.

    Query parameters:

    Name: String
    Width: Number
    Height Number

    How to write request:

    localhost:3000/api/image/resize?name=fjord&width=700&height=500
    localhost:3000/api/image/resize?name=encenadaport&width=200&height=300
    localhost:3000/api/image/resize?name=palmtunnel&width=500&height=400
